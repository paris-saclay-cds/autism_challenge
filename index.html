What is this challenge?

Autism spectrum disorder (ASD) is a developmental disorders affecting communication and behavior with different range in severity of symptoms. ASD has been reported to affect approximately 1 in 166 children.

Although there is a consensus on a relation between ASD and atypical brain networks and anatomy, those differences in brain anatomy and functional connectivity remain unclear. To address these issues, study on large cohort of subjects are necessary to ensure relevant finding.

Details on the data

Structural MRI (sMRI)

For each subject, a set of structural features have been extracted and stored in a CSV file. Each row corresponds to a subject and each column to a specific features. The features extracted are:

  • Normalized brain volume computed using subcortical segmentation of FreeSurfer;
  • Cortical thickness and area for right and left hemisphere of FreeSurfer.

Functional MRI (fMRI)

For each subject, we preliminary extracted signals using different brain parcellations and atlases and accounting for motion correction and confounds. Those brain atlases and parcellations are:

  • BASC parcellations with 64, 122, and 197 regions ( Bellec 2010);
  • Ncuts parcellations (Craddock 2012);
  • Harvard-Oxford anatomical parcellations;
  • MSDL functional atlas (Varoquaux 2011);
  • Power atlas (Power 2011).
The Nilearn documentation gives more information on the parcellations.

The time series for each subject are stored in a CSV file. Each row represent a scan while each column a region defined by the brain parcellation or atlas used.

Multimodal: assembling sMRI and fMRI data together

We provide the training data, stored in a single CSV file, training.csv.

It contains anatomical features and the path to the time series for a specific brain atlas. The time series data are downloaded on the fly by calling download_data.py atlas_name, avoiding the need to download all fMRI data before to start hacking.

Refer to the starting kit for more information.
